# 5-fiction-books
# Final project

5 books of the genre fiction Version 1.1 24/04/2016

The idea of this program is to show the best books of the genre fiction and to attract readers to read these books.

Идея программы состоит в том, чтобы показать именно те лучшие фантастические книги, которые стоит прочитать каждому. Как мне кажется, все, кто прочтут любую из этих книг, найдут в ней что-то свое. Надеюсь, подборка книг сможет помочь тем людям, кому сложно выбрать книгу к прочтению и, увидев это приложение, сразу выберут интересную и увлекательную книгу.
